# alrdy-animate

Attribute-driven scroll-animation and interactive-component library for Webflow and Next.js. GSAP is a peer dependency.

**Docs, install snippets, and live demos: <https://animate.alrdy.de>**

For AI coding agents working in downstream projects, see [`AGENTS.md`](./AGENTS.md).

> v8 is currently in alpha. For v7 (previous stable), install `alrdy-animate@latest` or see the [`v7-maintenance`](https://github.com/ben-alrdy/alrdy-animate/tree/v7-maintenance) branch.
