Copied! Switch to Webflow Designer and paste (⌘V / Ctrl + V).
  1. Click Copy to Webflow above. The JSON envelope is written to your clipboard with both application/json and text/plain MIME types.
  2. Switch to the Webflow Designer. Click on the canvas where the button should land.
  3. Paste (⌘V / Ctrl + V). Webflow reconstructs the element, classes, and base styles. The text span comes pre-wired with the custom attribute aa-split="chars index".
  4. Open Project Settings → Custom Code → Head and paste the contents of styles.html from this folder. Save and publish.
Works in Chrome and modern Chromium-based browsers. If pasting fails: don't have the same Webflow project open in two tabs (it corrupts the clipboard state). Re-click Copy to Webflow if your clipboard contents got replaced in between.
