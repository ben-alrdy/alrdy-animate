Slider Progress Bar Test

This test demonstrates the new aa-slider-progress feature. Progress bars are placed inside aa-slider-button elements and animate during the autoplay delay period for the active slide only.

How it works: Each slider button contains a progress bar that fills up when that slide is active. The progress animation uses the slider's aa-delay and aa-ease values.
