Scroll Tracking

The current indicator automatically follows the active section. Notice how the nav adapts its colors based on the section background.

Flip.fit(indicator, activeNavItem, {
  duration: 0.4,
  ease: 'power2.out',
  absolute: true,
  scale: true,
  simple: true
});

Hover Interaction

Try hovering over the navigation links above. The hover indicator originates from the current position and returns when you move away - creating a smooth, elastic effect.

mouseenter: Flip.fit(hover, item)
mouseleave: Flip.fit(hover, current)

// Returns to current indicator!
