`rotate` fades the element in from opacity 0 while tilting it from a 5┬░ offset back to its natural rotation. Add a corner suffix (`-tl`, `-tr`, `-bl`, `-br`) to pivot from that corner, `-ccw` to flip the direction, and use `rotate-up` to combine the rotation with the same upward translate as `fade-up`.

The animation is built with `gsap.from()`, so any rotation, opacity, or transform you set in CSS becomes the destination ŌĆö a card with a permanent `rotate(8deg)` will land back on `8deg`, not `0`.

## Variants

| Variant              | What it does                                                          |
| -------------------- | --------------------------------------------------------------------- |
| `rotate`             | rotates clockwise into place + opacity 0 ŌåÆ natural                    |
| `rotate-ccw`         | rotates counter-clockwise into place                                  |
| `rotate-tl` ┬Ę `-tr` ┬Ę `-bl` ┬Ę `-br` | pivots from the named corner instead of center        |
| `rotate-up`          | combines rotation with `fade-up`'s translate (3rem ├Ś distance)        |
| `rotate-up-bl-ccw`   | every modifier composes ŌĆö pick a corner, a direction, optionally up   |

## Attributes

| Attribute     | Default      | Notes                                                                  |
| ------------- | ------------ | ---------------------------------------------------------------------- |
| `aa-animate`  | (required)   | one of the variants above                                              |
| `aa-distance` | `1`          | multiplier on the 5┬░ angle (and on the 3rem y-offset for `rotate-up`)  |
| `aa-duration` | (init)       | seconds ŌĆö falls back to `init({ duration })` (default `0.6`)           |
| `aa-delay`    | `0`          | seconds                                                                |
| `aa-ease`     | (init)       | any GSAP ease ŌĆö falls back to `init({ ease })` (default `power4.out`)  |
| `aa-stagger`  | (off)        | when set on a parent with children, staggers the children              |
| `aa-scrub`    | (off)        | `true` or seconds ŌĆö bind progress to scroll                            |

## Required GSAP plugins

`gsap`, `ScrollTrigger`.
