title: slide
description: Translate-only entrance. Element slides in from a direction at full opacity — useful when you don't want a fade.
---

`slide-up`, `slide-down`, `slide-left`, `slide-right` translate the element into place from the corresponding direction. **No opacity tween** — the element is visible the whole time. Pair with `overflow: hidden` on a parent if you want a clipping reveal effect.

For an entrance that *also* fades, use [fade-up/down/left/right](/animations/appear/fade/).

## Variants

| `aa-animate` | Starts at                |
| ------------ | ------------------------ |
| `slide-up`   | translateY(100px × distance) |
| `slide-down` | translateY(-100px × distance) |
| `slide-left` | translateX(100px × distance) |
| `slide-right`| translateX(-100px × distance) |

## Attributes

| Attribute     | Default      | Notes                                                     |
| ------------- | ------------ | --------------------------------------------------------- |
| `aa-animate`  | (required)   | one of the variants above                                 |
| `aa-distance` | `1`          | multiplier on the 100px offset                            |
| `aa-duration` | `0.6`        | seconds                                                   |
| `aa-delay`    | `0`          | seconds                                                   |
| `aa-ease`     | `power4.out` | any GSAP ease                                             |
| `aa-stagger`  | (off)        | when set on a parent with children, staggers the children |
| `aa-scrub`    | (off)        | `true` or seconds — bind progress to scroll               |

## Required GSAP plugins

`gsap`, `ScrollTrigger`.
