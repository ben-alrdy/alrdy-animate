`blur-in` fades opacity 0 → 1 while the CSS blur filter shrinks from 20px to 0. The result is a soft, hazy entrance — useful for hero imagery and editorial layouts.

For per-character blur on text, see [text-blur](/animations/text/text-blur/) instead.

## Attributes

| Attribute     | Default      | Notes                                                     |
| ------------- | ------------ | --------------------------------------------------------- |
| `aa-animate`  | (required)   | `blur-in`                                                 |
| `aa-duration` | `0.6`        | seconds                                                   |
| `aa-delay`    | `0`          | seconds                                                   |
| `aa-ease`     | `power4.out` | any GSAP ease                                             |
| `aa-stagger`  | (off)        | when set on a parent with children, staggers the children |
| `aa-scrub`    | (off)        | `true` or seconds — bind progress to scroll               |

## Required GSAP plugins

`gsap`, `ScrollTrigger`.
