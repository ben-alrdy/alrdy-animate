A simple entrance: opacity 0 → 1 while translating from below into place. The most-used animation in production.

## Attributes

| Attribute     | Default     | Notes                                  |
| ------------- | ----------- | -------------------------------------- |
| `aa-animate`  | (required)  | `fade-up`                              |
| `aa-duration` | `0.6`       | seconds                                |
| `aa-delay`    | `0`         | seconds                                |
| `aa-ease`     | `power4.out`| any GSAP ease                          |
| `aa-distance` | `1`         | multiplier on the translate distance   |

## Required GSAP plugins

`gsap`, `ScrollTrigger`.

## Live demo

This demo exercises both responsive variants. Resize your browser and watch the elements switch animations as you cross 768px (`md`).
