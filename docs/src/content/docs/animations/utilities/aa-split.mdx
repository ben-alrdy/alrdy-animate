`aa-split` splits an element into `.aa-word`, `.aa-char`, or `.aa-line` parts and stops there. Use it when you want a CSS-only stagger, a hover effect, or you're driving the animation yourself. If GSAP's SplitText plugin is loaded, it's used for accuracy; otherwise words/chars fall back to a tiny regex splitter (`lines` requires SplitText).

## Attributes

| Attribute   | Values                          | Notes                                   |
| ----------- | ------------------------------- | --------------------------------------- |
| `aa-split`  | `words`, `chars`, `lines`       | with `\|` shorthand and suffixes        |

## Live demo

The headline below is split into words. Hover any word — pure CSS picks up the `.aa-word` class to apply a per-word transform.

<Demo>
  <div data-demo-stage>
    <h2 class="split-demo" aa-split="words">
      Hover any word in this headline.
    </h2>
  </div>
</Demo>
